Component library Icons

Icons are small visual elements, which can help show a change in something or signpost a call to action.

This component is not currently in the GOV.UK design system, but is in their GitHub backlog.

Icons are used across all of our services. It can be found on view latest funding and view national funding formula for schools.

MYESF currently use the following icons:

  • calculation
  • document
  • variance
  • print

When to use this component

Icons can be used as part of a larger component to help identify its purpose, such as the download a document or action link component.

Icons can also be used to help with understanding data, such as the increase and decrease indicators used on view national funding formula for schools. However, we must be careful to think about accessibility when using icons to convey important information.

When not to use this component

There is no standard icon to use for certain things. We must only use an icon when we are sure that the majority of users understand it to mean the same thing. There should also be accompanying text to help users understand what they are looking at.

We must use alternative text for assistive technology users. The call out must clearly describe what the icon means.

How it works

Calculation

The purpose of this component is to give users an implicit visual cue as to the relationship between two numerical figures which make up a total.

It should only be used when the use of ordering in a table, content or other visual cues are not enough to help users understand the relationship between two figures.

Variance

We sometimes need to inform users that there has been a change in the funding data they are viewing - we call this variance. When showing variance to users we add the information as closely to the data it relates to so users can more easily understand what it means.

  • are in online icon libraries, which can be linked within the custom CSS folder
  • are in the prototype images folder (/public/images)

Research on this component

Users have almost unanimously found icons to be useful elements in drawing meaning from information presented on a page.

Where we have displayed funding data for a period, icons indicating whether the figure has increased or decreased between the current and previous periods enables users to get a quick sense of variance.

Icons associated with, for example, links to download documents, have acted as effective signposts to the action.
