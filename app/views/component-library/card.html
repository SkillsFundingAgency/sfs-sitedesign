Component library Card

Panels of structured content - for example, image, title, description, link. They are typically organised in rows or grids.

This component is not currently in the GOV.UK design system, but is in their GitHub backlog.

The card component is used as a form of navigation on the MYESF dashboard and as a way to display data on view national funding formula for schools.

When to use this component

Cards can be a good way to display related content, data and links. Use the cards to display items on a page if the data is simple to understand.

If cards are organised in a row or a grid, there is no hierarchy. This means they can be useful for dashboard pages where there are a number of journeys to select.

When not to use this component

Do not use this component for complex data. For example, when long explanations are needed to help users understand the relationship between data points. Users should be able to easily understand the relationship by looking at the data type and value.

How it works

As navigation

For cards used in navigation, such as the MYESF dashboard:

  • the top section of the card is used to identify the next page
  • the bottom section is dynamic and gives more information about what to expect on the next page

To display data

For cards used on statement pages, such as on view national funding formula for schools:

  • the top section of the card displays the data type and its value
  • the bottom section is for extra content to provide contextual help if users need it

To display data - one third width

If users need contextual help, the content in the bottom section should be less than 50 characters.

Research on this component

Confirmation can be taken from testing that users give the card component more 'weight' than other means of presenting data.

They have been more proactive in studying the information presented, and a card is associated with interactivity.

Cards have been observed to be particularly helpful to users when communicating variance, as per the example given.
