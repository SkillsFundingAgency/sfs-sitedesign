Component library Signed-in account banner
New to the Design System

A banner which provides information and navigation options to the user on a signed-in service.

The signed-in account banner is used across all signed in areas of MyESF dashboard.

Open this signed-in account banner example in a new window

  • CSS
HTML
CSS

When to use this component

Use this component to help a user understand that they have signed in to our service. This can include metadata about the organisation or user profile they have signed in as.

This can also be accompanied by navigational links to other relevant parts of the service that need to be accessed throughout the users journey.

When not to use this component

Do not use this component on public services where a user has not entered any credentials to identify themselves.

How it works

You must provide a way for the user to log back out when they have signed in, this should be placed in a familiar position as to not get the user lost.
