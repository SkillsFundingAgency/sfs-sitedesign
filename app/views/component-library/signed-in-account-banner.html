Component library Signed-in account banner
New to the Design System

A banner which provides information and navigation options to the user on a signed-in service.

The signed-in account banner is used across all signed in areas of MYESF dashboard.

When to use this component

Use this component to help a user understand that they have signed in to our service. This can include details about the organisation or user profile they have signed in as.

This can also be accompanied by navigational links that can quickly take users to other parts of the service.

When not to use this component

Do not use this component on public services where a user has not entered any private information to identify themselves.

How it works

You must provide a way for the user to log back out when they have signed in. This should be placed in a familiar position to help the user find it easily.

Research on this component

This component clarifies to a user whether or not they are signed in, and it is understood in addition as their path to sign out.

Prior research with the signed-in account banner has highlighted its use in informing users of account specific permissions or limitations, based, for example, on the individual user's assigned roles.
