Component library Funding header

A custom styled typography which is used to give headline funding totals within a top level heading on the page.

View more general guidelines on this style in the GOV.UK design system.

The funding header section is used as a way of introducing totals for funding types in view latest funding and on MYESF allocation statements.

When to use this component

This component can be used to show totals to users within top level headings on a page.

When not to use this component

Do not use this component on the same page as a lot of other headings of similar size. This can confuse the user and reduce its visibility on the page.

This component should only be used once on a page. The only time to use it more than once is on separate GOV.UK Design System tabs.

How it works

Full width

Use this version of the component as a heading when displaying the total funding for a local authority, school or college. For example, on MYESF allocation statements we use this version to display the total funding amount in the <h1>.

Float left

Sometimes we need to use tabs to show how total funding is broken down into funding type. For example, we use tabs to break down the DSG into different funding types in view latest funding. To meet user needs, we display the DSG total funding heading amount as well as the totals for each funding type.

When using this version of the component, make sure that the user can print each tab by including a print this page action link on the right hand side of the funding header.

Research on this component

As long as there is no ambiguity for the user around what this headline figure represents, research supports the need of users for presenting summary figures boldly, so as to give immediate context to any further information.
