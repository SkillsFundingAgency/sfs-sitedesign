Component library Funding header

A custom styled typography which is used to give headline funding totals within a top level heading on the page.

View more general GDS guidelines on this style in the GOV.UK design system.

The funding header section is used as a way of introducing totals for funding types in view latest funding and on MyESF allocation statements.

Open this funding header example in a new window
  • HTML
  • CSS
HTML 
<h2 class="lede">
  Total 2018 to 2019 allocation
  <p class="heading-large heading-funding-summary">£441,116</p>
</h2>
CSS 
.heading-funding-summary {
  margin-top: 5px;
}

When to use this component

This component can be used to show totals to users within top level headings on a page.

When not to use this component

Do not use this component on the same page as a lot of other headings of similar size. This can confuse the user and reduce its visibility on the page. Also this component should not be duplicated on the same page.

Only show one instance of this component per page, unless the GOV.UK Design System tabs component is being used as part of the same page.

How it works

Full width

When displaying the total for an entire funding level, section or block then use this version of the component to display a funding total as a heading. For example, on MyESF allocation statements we use this full width version of the component to display the total funding amount in the <h1> on that page.

  • CSS

</h2>

Float left

Sometimes we need to use tabs to show a breakdown of a funding level, section or block for a funding type. For example, we use tabs to breakdown the funding blocks of the DSG on view latest funding. To meet user needs we display the DSG total funding heading amount as well as the totals for each of the child funding blocks.

Open this funding header example in a new window
  • HTML
  • CSS
HTML 
<div class="grid-row">
  <div class="block-header-total">
    <h2 class="lede">
      Total 2018 to 2019 allocation
      <p class="heading-large heading-funding-summary">£441,116</p>
    </h2>
  </div>
  <div class="block-header-print block-header-print-tabs">
    <a class="print-page" href="#"><i class="fa fa-print" aria-hidden="true"></i>Print this page</a>
  </div>
</div>
CSS 
.block-header-print-tabs {
  display:"none";
}

@media (min-width: 641px) {
  .block-header-print-tabs {
    display:"inherit";
  }
}

.fa {
  padding:"0 0.3em 0 0";
}

.block-header-total {
  padding:"0 15px";
  -webkit-box-sizing:"border-box";
  -moz-box-sizing:"border-box";
  box-sizing:"border-box";
}

@media (min-width: 641px) {
  .block-header-total {
    float:"left";
    width:"78%";
  }
}

.block-header-print {
  padding:"0 15px";
  -webkit-box-sizing:"border-box";
  -moz-box-sizing:"border-box";
  box-sizing:"border-box";
}

@media (min-width: 641px) {
  .block-header-print {
    text-align:"right";
  }
}

.heading-funding-summary {
  margin-top: 5px;
}

When using this version of the component make sure that the user can also print the breakdown within the tab by including a print this page action link to the right hand side of the funding header.
