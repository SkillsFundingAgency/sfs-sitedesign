Component library Links

This is an expansion on the GOV.UK Elements link component.

View more general guidelines on this style in the GOV.UK Elements. Once MYESF updates its core CSS libraries from GOV.UK Elements to the GOV.UK Design System, this component will be updated accordingly.

Links are used across all of our services. They can be found on MYESF start page, view latest funding and view national funding formula for schools.

  • HTML
HTML 
<a href="#">Allocation history</a>

When to use this component

Use this component when users need digital resources not already on their current page.

When not to use this component

If users need a link, then provide the most suitable type of link to satisfy the related need. For example, do not use a link to external content for an internal service link.

How it works

Standard

Use this type of the component when linking users to a digital resource or page within the same service.

Open this links example in a new window
  • HTML
HTML 
<a href="#">Allocation history</a>

External

When linking to anything hosted outside of the service, this is deemed an as external resource and therefore this link type should be used instead.

Open this links example in a new window
  • HTML
HTML 
<a href="https://www.gov.uk/government/publications/guide-to-national-funding-formula" rel="noreferrer noopener" target="_blank" title="Opens in a new tab">National funding formula: quick guide</a>

To page with tabs

Use this type of the component when linking directly to funding breakdown pages which also have GOV.UK Design System tabs present.

When using this component:

  • only use it when linking to a funding breakdown page which contains tabs
  • include a link for each of the available tabs on the target page
  • ensure the clicked link takes users to the target page with that corresponding tab selected on load

With action

We use this type of the component when presenting users with a link which performs a functional task for them. For example, when printing a page.

Some of the action links need to be combined with an accompanying icon, such as the print this page action link which always needs to be implemented with the print icon.

Open this links example in a new window
  • HTML
  • CSS
  • JavaScript
HTML 
<div class="print-a-document-container">
  <div class="print-icon">
    <i class="fa fa-print fa-print-large"></i>
  </div>
  <div class="print-a-document">
    <a class="print-page" href="#"><i class="fa fa-print" aria-hidden="true"></i>Print this page</a>
  </div>
</div>
CSS

The related print component should hide when a user opens the print dialogue. You will need to reference the following CSS resource below in the print.css file:

.print-page {
  display: none;
}

You will need to add the following CSS to style the action link:

.print-a-document-container {
  padding-bottom: 0px;
}

.print-icon {
  float: left;
  width: 2.5em;
}

.fa {
  padding: 0 0.3em 0 0;
}

.fa-print-large {
  font-size: 28px;
}

@media (min-width: 641px) {
  .fa-print-large {
    font-size: 34px;
  }
}

.print-a-document {
  float: none;
  font-weight: 600;
  min-height: 2.5em;
  vertical-align: top;
  padding-left: 2.8em;
}
JavaScript

The print dialogue functionality uses the following JavaScript:

$(document).ready(function () {

  $(".print-page").click(function (e) {

    e.preventDefault();
    window.print();

  });

});

This print icon uses a third party JavaScript library to dynamically generate the optimum CSS for the device utilising it. You will need to reference the following JavaScript resource below in the HTML:

<script src="https://kit.fontawesome.com/71943611db.js" crossorigin="anonymous"></script>

Funding table guidance

Although this link type is very much like the external is differs due to it being reserved exclusively for use within a funding table.

This funding table guidance link type is related to the funding table with link component.

When using this component:

  • only use it when providing a link for users within a funding table
  • only include links in a funding table to help users understand why the related data is decimalised
  • ensure the link works like an external link and opens in a new tab or window

Research on this component

Users are familiar with the appearance of the links and have had no issue during testing identifying them, on the condition that they have been placed in an appropriate and accessible location on the page.

Links 'with action' have been observed by users as useful additions, as many have a need to print or download the data as a part of their process; disseminating to colleagues or for annotation purposes.
