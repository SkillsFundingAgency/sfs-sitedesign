Component library Organisation banner

A banner that sits below the global GOV.UK header to help a user identify which department they are interacting with on GOV.UK.

This component is not currently in the GOV.UK design system, but is in their GitHub backlog.

The organisation banner is used on view latest funding and view national funding formula for schools.

When to use this component

Use this component when it's useful for the user to know which organisation is providing the service, for example, ESFA. This means there is no need to identify ESFA on each page.

When not to use this component

For now, this component only works when there isn’t already a banner in that place. More design work is needed for services like MYESF. In these cases, the place is used to sign in and navigate to different pages.

How it works

The banner has been designed in-house. GDS and ESFA have approved it to be used across our services.

The banner can be implemented in the same way as the phase banner on the design system. If both the phase and ESFA banners are needed, they should be in the following vertical order:

  • GOV.UK header
  • phase banner
  • ESFA banner

Research on this component

The organisation banner has aided users in orienting themselves with the regard to the government ecosystem; in making it clearer which agency the page or service sits under, research participants have intimated that they have more confidence they are the correct place to perform their task.
