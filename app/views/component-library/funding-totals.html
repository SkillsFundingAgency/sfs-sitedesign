When to use this component

This component can be used to show important totals to users.

When not to use this component

Do not use this component on the same page as a lot of other headings of similar size. This can confuse the user and reduce its visibility on the page.

How it works

Grand total

When displaying the total for an entire funding level, section or block then use this version of the component to display a funding total. For example, on view latest funding we display the grand total for each block of the dedicated school grant with this type of funding total.

This is the default design type of this component to use for displaying funding totals figures to end users in MYESF. Use this type whenever possible for displaying funding totals to users in the service.
