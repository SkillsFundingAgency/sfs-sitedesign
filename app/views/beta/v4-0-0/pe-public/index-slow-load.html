Multi-academy trusts (MATs) are academy trusts that run more than one school or college. Where there is more than one MAT under the same academy sponsor, we publish performance figures in the name of the sponsor as if it were a single MAT. Selecting either the MAT or sponsor name, which are often similar, will take you to the same information.
