Total allocation before deductions: £117,596,000
How is this total calculated?

The primary unit of funding (PUF) and secondary unit of funding (SUF) for each local authority are multiplied by:

  • PUF from the October 2018 school census
  • SUF from the October 2018 school census

We then add:

  • the actual 2019 to 2020 funding for the premises and mobility factors
  • growth funding using the differences between the primary and secondary numbers on roll in the local authority on the October 2017 and October 2018 school censuses

For more information on these calculations, you can visit the dedicated schools grant: technical note 2019 to 2020.

How is this total calculated?

We will calculate amounts for recoupment using data provided on the authority proforma tool (APT) submitted in January 2019.

This its then subracted from the total initial schools block allocation to get a total schools block allocation after deductions.

For more information on these calculations, you can visit the dedicated schools grant: technical note 2019 to 2020.
