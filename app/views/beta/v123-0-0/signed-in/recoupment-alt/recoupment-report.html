How recoupment for this opening period is calculated

We calculate recoupment to allow for a 20% rate relief by using the following equation:

(Post de-delegation budget − Rates) + (Rates × 0.2) = Full year recoupment

We then pro-rata this for the days the academy has been open:

(Full year recoupment ÷ 365) × Days open = Recoupment adjusted pro rata

We finally add the de-delegated funding pro-rata:

(Recoupment adjusted pro rata+7/12 of de-delegation) +/− Net adjustment = Total recoupment
